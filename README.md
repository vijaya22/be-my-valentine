# Be My Valentine

A simple, static website to ask someone to be your valentine. Create a personalized valentine card and share the link with your special someone.

**Live site:** [vijaya22.github.io/be-my-valentine](https://vijaya22.github.io/be-my-valentine/)

## How it works

1. Visit the site and fill in your name, their name, and an optional message
2. Pick a color theme (Classic, Rose, Sunset, or Royal)
3. Click "Generate Valentine Link" and share the link
4. When they open the link, they see a personalized valentine card
5. They can click **Yes** (confetti!) or try to click **No** (good luck with that)

## Features

- Personalized valentine cards via URL — no backend needed
- 4 soft pastel color themes with live preview
- Pre-written message suggestions with a shuffle button
- The "No" button dodges around the screen when you try to click it
- Confetti animation on clicking "Yes"
- Typewriter-style font on a cream paper background
- Fully responsive — works on mobile and desktop
- Hosted free on GitHub Pages

## Tech stack

- Vanilla HTML, CSS, and JavaScript
- No frameworks, no build tools, no dependencies
- Google Fonts (Special Elite)
- GoatCounter for privacy-friendly analytics

## Run locally

Just open `index.html` in a browser. That's it.
